## How to Take Screenshots

To capture screenshots of your ranch system:

1. Join your RedM server
2. Press `F5` to open the Ranch UI
3. Navigate to each tab
4. Use RedM's screenshot feature or a screen capture tool
5. Save screenshots with the appropriate filename
6. Place them in this `screenshots/` directory

## Image Specifications

For best results, screenshots should be:
- **Format:** PNG
- **Resolution:** 1920x1080 (Full HD) or higher
- **Aspect Ratio:** 16:9
- **File Size:** Under 2MB each
- **Content:** Clear view of the UI element being showcased

## Creating Professional Screenshots

Tips for high-quality screenshots:
1. **Lighting:** Use good in-game lighting conditions
2. **Angle:** Center the UI in the frame
3. **Sample Data:** Populate the UI with realistic ranch data
4. **Clean UI:** Hide unnecessary elements or overlays
5. **Consistency:** Use the same resolution for all screenshots
6. **Annotations:** Add arrows or highlights to emphasize features (optional)

## Video Demonstrations

For video demonstrations:
- Record at 1080p 60fps minimum
- Show navigation between tabs
- Demonstrate key features in action
- Keep videos under 2 minutes for each feature
- Upload to YouTube or similar platform
